## 4. 트러블 슈팅
### 웹페이지 데이터 추출 오류
* 문제 : BeautifulSoup를 이용하여 데이터 수집하던 중 제품의 가격이 모두 None으로 나오는 오류가 발생했습니다.
* 해결 : Selenium를 이용하여 동적 페이지에서도 원하는 데이터를 추출할 수 있게 되었습니다.
